## рҹ©ә What is AlagaHub?

**AlagaHub** is a cross-platform mobile application built with Flutter that connects **patients** with **healthcare workers** in a streamlined, accessible way. Whether you're booking a consultation, messaging your doctor, or managing your health profile вҖ” AlagaHub brings it all into one place.

> *"Alaga"* (Filipino) вҖ” to care for, to tend to, to look after.

## рҹҡҖ Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (3.x or later)
- [Dart](https://dart.dev/get-dart)
- [Firebase CLI](https://firebase.google.com/docs/cli)
- An Android or iOS device / emulator

### Installation

**1. Clone the repository**
```bash
git clone https://github.com/zekeyell/AlagaHub.git
cd AlagaHub
```

**2. Install dependencies**
```bash
flutter pub get
```

**3. Set up Firebase**

> вҡ пёҸ Firebase config files are excluded from this repo for security.

- Go to [Firebase Console](https://console.firebase.google.com/)
- Create a new project
- Add Android and/or iOS apps
- Download `google-services.json` вҶ’ place in `android/app/`
- Download `GoogleService-Info.plist` вҶ’ place in `ios/Runner/`
- Run `flutterfire configure` to generate `firebase_options.dart`

**4. Run the app**
```bash
flutter run
```
